HOUSE-OF-STARK

This website deciphers the hidden messages sent to Jon Snow through an immersive **Game of Thrones** inspired experience.

STRUCTURE

**Letter for Jon Snow → Landing Page → Enter Hidden Message → Decipher Hidden Message → Display Result → Return to Landing Page → Repeat Process**

THEME

**Game of Thrones – House Stark**


FEATURES
Landing Page

* Animated Direwolf sigil with glow effect
* Tagline: **"Winter is Coming... and so is the truth."**
* House Stark inspired dark theme with parchment aesthetics
* **Begin Deciphering** button styled as a wax seal
* Ambient effects including falling snow and a raven flying across the screen
* Background music toggle (muted by default)
* **Beyond the Wall Mode** (dark mode toggle)

Letter for Jon Snow

* Interactive sealed envelope displayed on startup
* Wax seal opening animation
* Maester's introductory letter containing an encrypted message
* **Continue** button leading to the main application

📜 Message Entry Page

* Parchment-style textarea for entering the encrypted message
* Placeholder text: **"Paste the raven's message here..."**
* **Send to Maester** button with raven-themed transition
* Input validation to ensure a message is entered before proceeding

🔮 Deciphering Page

* Animated deciphering sequence with raven flight
* Progress bar styled to match the medieval theme
* Dynamic processing effects
* Flavor text displayed while the message is being deciphered
* Sound effects such as quill writing and raven calls (optional)

📖 Result Page

* Deciphered message displayed on an unrolled parchment scroll
* Animated **"Sealed by the Maester"** wax stamp
* Options to:

  * Copy the deciphered message
  * Download the result as a scroll (image/PDF)
  * Share the message (**"Send this Raven Onward"**)
* Random House Stark themed flavor line displayed beneath the result

🔁 Reset / Repeat Flow

* **Return to Winterfell** button returns the user to the landing page
* **Ravens Received** session history stores the latest deciphered messages locally (last five messages)
* Optional message counter showing the number of messages deciphered during the session

🎨 General Theme Features

* House Stark inspired color palette (grey, white, ice-blue, parchment tan)
* Custom direwolf paw cursor
* Continuous snowfall particle effects
* Responsive design for desktop, tablet, and mobile devices
* **Beyond the Wall Mode** dark theme toggle
* Medieval fonts, parchment textures, wax seals, and Game of Thrones-inspired visual styling
