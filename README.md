# HOUSE-OF-STARK
This website deciphers the hidden messages that was sent to Jon Snow

# STRUCTURE:
Landing page-> Entering the hidden message -> Deciphering the Hidden message -> Printing the result -> Going back again to the landing page to repeat the process again

#THEME : Game of Thrones

#FEATURES:
(Landing → Enter Message → Decipher → Result → Repeat):
🏰 Landing Page

Direwolf sigil animation on load (howl/glow effect)
Tagline: "Winter is Coming... and so is the truth"
Parchment/scroll-textured background with House Stark banner (grey/white, direwolf motif)
"Begin Deciphering" call-to-action button styled as a wax seal
Ambient touches: flickering torchlight, subtle raven flying across screen
Background music toggle (Game of Thrones-style theme, optional/muted by default)

📜 Message Entry Page

Textarea styled like a raven-scroll / parchment note
Placeholder text like "Paste the raven's message here..."
Option to select cipher type (e.g., Caesar shift, Valyrian rune substitution, reverse text, keyword cipher)
A "shift key" or "cipher key" input (numeric or House-sigil-based selector — e.g., choose a House whose numeric sigil value acts as the key)
"Send to Maester" button (submit) with a raven-flying animation transition

🔮 Deciphering Page

Animated "decoding in progress" sequence — e.g., raven flying, candle burning down, or runes unscrambling letter by letter
Progress bar styled as a melting wax seal or burning scroll
Optional flavor text/quotes cycling during processing (Stark-themed, original — not reproduced show dialogue)
Sound effect: quill scratching / raven cawing

📖 Result Page

Deciphered message displayed on an unrolled scroll
"Sealed by the Maester" stamp/badge once complete
Option to copy result or download as a scroll (image/PDF)
Share button ("Send this raven onward")
Fun touch: a random Stark motto flavor line based on message content length/tone (e.g., short = "Brace yourself", long = "The North Remembers")

🔁 Reset / Repeat Flow

"Return to Winterfell" button → navigates back to landing page
Session-based history: "Ravens Received" log showing past 5 deciphered messages (stored locally, not persisted across sessions)
Optional: small counter — "Messages deciphered: X"

🎨 General Theme Features

Color palette: House Stark grey, white, ice-blue accents, parchment tan
Custom cursor (direwolf paw or sword icon)
Subtle snow-falling particle effect in background
Responsive design (works on mobile/tablet — "even beyond the Wall")
Dark mode toggle labeled "Beyond the Wall Mode"
